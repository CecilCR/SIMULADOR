🎯 Mapa SCARF orientativo

El modelo SCARF (Rock, 2008) identifica cinco dominios sociales que el cerebro puede leer como recompensa o amenaza. Aquí los porcentajes son indicadores heurísticos para la reflexión didáctica. Selecciona cada dominio para interpretar qué puede haber activado tu respuesta.

Valores más altos sugieren mayor activación percibida en ese dominio dentro de esta simulación formativa.
